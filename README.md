# Android使用入门与进阶

这里讲的是原生Android，之后可能会涉及IOS

## 刷机步骤

1. 解锁bt锁
2. 刷入Recovery，这里推荐[TWRP](<https://twrp.me/Devices/>)
3. 刷入第三方刷机包，这里一般都是卡刷包。
4. 刷入[Gapps](<https://opengapps.org/>)
5. 刷入Root包，不Root的Android与咸鱼有啥区别？？？
6. 美化与优化
7. 线刷上锁回归（hah）
